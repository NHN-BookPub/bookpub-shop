== 개요

이 API 문서는 Book-Pub 팀의 Api 문서를 유용하게 볼수있게 도와줍니다.

[NOTE]
====
해당 프로젝트는 API 문서는 개발환경까지 노출합니다.

기본적인 API 구성은

    | 1. /token/ ** |

    |  2. /api/ ** |

총 2가지로 구성되어 있습니다.
====

=== API 서버 경로

|====
|환경         | DNS|비고
|SHOP 개발(dev)    | |
